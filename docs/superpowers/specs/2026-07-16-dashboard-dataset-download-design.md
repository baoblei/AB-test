# Dashboard 测评集下载与布局优化设计

## 目标

在现有统计看板中增加单个测评集下载能力，并解决管理区拥挤、长文本显示不全的问题。保留现有统计总览、坏例和导出功能的层级与行为。

## 已确认的产品规则

- 每次只能选择并下载一个测评集。
- T2I 始终只下载对应场景的 prompt TXT。
- TI2I 默认不勾选“包含参考图”。
- TI2I 未勾选参考图时下载 prompt TXT；勾选后下载 ZIP。
- TI2I ZIP 包含 prompt TXT 和 `ref_images/` 目录。
- 测评集可通过搜索框过滤，也可直接从下拉框选择。

## Dashboard 布局

页面采用水平分层结构，按以下顺序纵向排列：

1. 顶部导航。
2. “发布新测试任务”层，内部并列展示“上传新测评集”和“上传新结果图”。
3. 独立的“下载测评集”层。
4. 独立的“统计筛选”层。
5. 模型对战总览层。

桌面端通过自适应网格分配字段宽度，输入框和按钮不挤压长标签。中等宽度时字段分两列折行，窄屏时改为单列。文字允许合理换行，选择框中的超长值使用省略号并保留完整值提示。

## 下载交互

下载层包含任务类型、搜索框、测评集下拉框、TI2I 参考图复选框和下载按钮。

- 任务类型默认与当前看板类型一致，切换后重新加载对应测评集。
- 搜索框在浏览器中即时过滤已加载的测评集列表，不在每次输入时请求后端。
- 下拉框单选一个测评集，并显示场景名称与 prompt 条目数。
- T2I 隐藏或禁用参考图复选框，按钮文案为“下载 TXT”。
- TI2I 默认不勾选参考图；未勾选时按钮文案为“下载 TXT”，勾选后为“下载 ZIP”。
- 下载期间禁用按钮并显示生成状态；成功后恢复，失败时在下载层内显示明确错误。

## 现有总览行为

不改变模型对战总览的信息结构：

- 全场景汇总栏继续展示各维度百分比、数量和 A/B 双向压制比。
- TI2I 继续额外展示保真度维度。
- 全场景汇总栏保留“统计”“坏例详情”“导出”。
- 各场景统计默认折叠，通过汇总栏或展开箭头打开。
- 每个场景保留“明细”“统计”“坏例”，并显示对应维度结果、压制比、坏例占比和高频坏例。

## 后端设计

新增两个受登录保护的接口：

1. 测评集列表接口：按规范化后的 `task_type` 返回场景名称和 prompt 条目数。
2. 单场景下载接口：接收 `task_type`、`scene` 和 `include_ref`。

下载行为：

- T2I 忽略或拒绝不适用的参考图请求，并返回 prompt TXT。
- TI2I 且 `include_ref=false` 时返回 prompt TXT。
- TI2I 且 `include_ref=true` 时，在临时目录中生成 ZIP；根目录放置 prompt TXT，参考图放入 `ref_images/`。
- smoke 数据同样使用 `prompt/<task_type>/<scene>.txt`；TI2I smoke prompt ID 必须与 `ref_images/TI2I/<scene>` 完整对应，不能复用 T2I prompt。
- 文件名使用安全、可预测的场景名称，并通过 `Content-Disposition` 返回 UTF-8 下载名。
- ZIP 响应结束后清理临时文件。

服务逻辑放在存储或独立下载服务中，路由只负责参数接收、登录校验和文件响应，不把打包细节写入 `main.py`。

## 校验和错误处理

- `task_type` 仅允许 T2I 和 TI2I。
- 场景名称使用现有安全路径规则校验，拒绝绝对路径、父目录跳转和符号链接逃逸。
- prompt 文件不存在时返回明确的业务错误。
- TI2I 勾选参考图后，参考图目录必须存在，且图片 ID 必须与 prompt ID 完整对应；否则拒绝生成残缺 ZIP。
- 空测评集列表、列表加载失败和下载失败均在对应卡片内显示，不影响统计看板其余功能。

## 测试与验收

后端自动化测试覆盖：

- T2I 和 TI2I 测评集列表及条目数。
- T2I TXT 下载。
- TI2I 未勾选参考图的 TXT 下载。
- TI2I 勾选参考图后的 ZIP 目录和内容。
- 无效任务类型、非法场景路径、缺失 prompt、缺失或不完整参考图。
- 路由登录保护、响应类型、下载文件名和临时文件清理。

前端自动化测试覆盖：

- 下载层所需控件和任务类型联动。
- 搜索过滤与单选行为。
- TI2I 参考图默认不勾选，T2I 不提供参考图下载。
- 下载按钮状态、文件名解析和错误展示。
- 管理区、下载区、统计筛选区水平分层。
- 全场景与场景级操作按钮保持不变。
- 各场景默认折叠，各维度压制比仍显示。

人工验收在桌面和窄屏宽度下检查字段不重叠、重要文本完整、操作层级清晰，并分别实际下载 T2I TXT、TI2I TXT 和包含参考图的 TI2I ZIP。
